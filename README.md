# Unlock All Maps

A Slime Rancher 2 mod that unlocks all maps.

## Installation

1. Install [Melon Loader](https://melonwiki.xyz/#/?id=automated-installation) for Slime Rancher 2
1. Download the [latest release](https://github.com/markekraus/SR2UnlockAllMaps/releases)
1. Copy `UnlockAllMaps.dll` to the `Mods` folder in your Slime Rancher 2 install folder (i.e. `C:\Program Files (x86)\Steam\steamapps\common\Slime Rancher 2\Mods`)
